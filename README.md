# Exercice-pratique-final-badjolao-gmail.com-sql
Cette base de données gère les activités d’un centre informatique au Togo. Elle permet d’enregistrer les clients, les techniciens, les services proposés et les interventions réalisées. Chaque intervention relie un client, un technicien et un service avec la date, la durée et le montant facturé afin d’assurer un suivi efficace des prestations.
